# Development

# Requirements

* node.js 8.0+
* npm 5.0+

*NOTE:* we recommend using nvm
```sh
brew install nvm
nvm install 8
nvm use 8
```

## Setup

```sh
git clone https://github.com/brianneisler/moltres.git
cd moltres
npm install
```


## Build

To build moltres

```sh
npm run build
```


## Clean

To clean moltres

```sh
npm run clean
```


## Watch

To watch moltres and build on every file change

```sh
npm run watch
```

## Test

To run tests against moltres

```sh
npm test
```
